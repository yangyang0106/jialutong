## 当前架构原则

- 老人端只负责路线执行体验，不负责算路、抽锚点、审核规则。
- 小程序家属端负责采集、审核和发布操作，不保存路线引擎兼容实现。
- 后端是路线资产、路线引擎、AI、TTS、审核、行程结果的唯一权威入口。
- 百度原始路线到家路通 RouteStep 的转换只允许在后端完成。
- AI 只能生成建议，不能发布路线。
