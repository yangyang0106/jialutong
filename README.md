# 家路通微信小程序 MVP

这是一个纯前端微信小程序 MVP。没有后端、登录、数据库或 AI，路线数据保存在 `data/routes.js`。

当前固定路线主干为：

富友嘉园一期 → 步行或骑行至临洮路站 → 14 号线 → 曹杨路换乘 3 号线 → 江湾镇站 → 887 路 → 步行至彩虹湾墨翠里。

步行和骑行段必须将每一个转弯、过红绿灯、进出站动作拆成独立步骤。

## 文盲老人模式

- 步骤进入后自动播放本地真人语音，并每 20 秒重复一次。
- 连续两次进入目标点 30 米范围后自动进入下一步。
- 持续远离目标点时震动并显示“请停下，不要继续走”。
- 求助界面向路人展示目的地、当前步骤和当前位置。
- 路线页提供大号“上一条”按钮。
- 图片可配合 `direction` 显示醒目的方向箭头。

## 路线图片和真人语音

当前有 23 个步骤：去程 13 步、回程 10 步。

- 最稳妥配置：23 张步骤照片、23 段真人语音。
- 首页进入“家属配置路线素材”，可逐步拍摄照片、录音、试听和删除。
- 素材通过 `wx.saveFile` 保存在当前手机的小程序本地文件中，不会上传到服务器。
- 删除小程序、清理数据或更换手机后，本地素材可能丢失。正式产品应增加云存储或后端上传。

### 可选上传接口

在 `config/upload.js` 中填写 `uploadUrl` 后，家属配置页会改用 `wx.uploadFile` 上传素材。

- 请求方式：`multipart/form-data`
- 文件字段：`file`
- 附加字段：`routeId`、`stepNo`、`kind`
- `kind` 值：`image` 或 `audio`
- 接口响应：`{ "url": "https://example.com/files/xxx.jpg" }`

微信小程序后台需要将上传服务器域名加入 uploadFile 合法域名。
可在 `config/upload.js` 中填写 `uploadToken`，上传时会发送 Bearer 鉴权头。

仓库内已提供独立的家路通服务：`../jialutong-server`。

部署后配置示例：

```js
module.exports = {
  uploadUrl: "https://your-domain.example.com/api/files",
  apiBaseUrl: "https://your-domain.example.com",
  uploadToken: "与服务端 JIALUTONG_UPLOAD_TOKEN 一致"
};
```

图片、语音、动作说明、方向和坐标都会同步到服务端。老人更换手机后，首页会自动拉取远程路线配置。

## 安全限制

- 路线未补齐照片、语音、动作说明、方向箭头、坐标和实地确认前，首页禁止启动。
- 低精度定位不会触发自动下一步或偏航判断。
- 小程序进入后台时会暂停轮询；返回后必须重新确认当前步骤。
- 行程进度保存在本机，重新进入路线时会恢复。
- 无后端版本无法自动通知家属或让家属远程查看位置，偏航时会停止路线并展示拨号求助界面。
- 本地模式会压缩照片，但 23 张照片和 23 段语音仍可能达到微信本地文件容量上限；正式使用应配置上传服务。
- 小程序内置的固定上传 Token 不能作为高安全鉴权方案；正式产品应由后端签发短期上传凭证。

## 本地运行

1. 打开微信开发者工具。
2. 选择“导入项目”，目录选择本文件夹。
3. 使用测试号或自己的小程序 AppID。
4. 在开发者工具和真机上允许定位权限。

## 高德地图接入

当前项目是原生微信小程序，不能直接使用浏览器 JavaScript API 或 `securityJsCode`。

1. 在高德开放平台创建“Web 服务”类型 Key。
2. 将 Key 填入 `config/amap.js` 的 `webServiceKey`。
3. 在微信小程序后台将 `https://restapi.amap.com` 加入 request 合法域名。
4. 使用 `utils/amap.js` 获取步行、骑行和公交路线。

已提供：

- `getWalkingRoute(origin, destination)`
- `getBicyclingRoute(origin, destination)`
- `getTransitRoute(origin, destination, city)`

坐标参数格式为 `经度,纬度`。高德返回的步行/骑行 `steps` 可用于生成每个转弯锚点。

## 上线前必须替换

- 家属必须打开高德地图并陪同实地完整走一次路线。
- 在 `data/routes.js` 中为每个步行或骑行转弯点填写明确的左转/右转提示和准确坐标。
- 为每个转弯点拍摄面向行进方向的照片，并填写对应 `image`。
- 为每个步骤录制真人语音并填写对应 `audio`。
- 在家属配置页确认家属电话和紧急求助电话。
- 确认临洮路站入口、曹杨路换乘方向、江湾镇站出口、887 路上下车站。
- 将地点照片放入例如 `assets/images/`，并填写每一步的 `image` 路径。
- 将音频放入例如 `assets/audio/`，并填写每一步的 `audio` 路径。
- 默认家属电话和紧急求助电话均为 `16621633647`，可在家属配置页修改。

定位每 5 秒刷新一次。当前位置距离当前步骤锚点小于 30 米时，页面会提示可以进入下一步。
